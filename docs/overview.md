# Overview

The Control Plane Framework v2 establishes a repository-native control layer for AI-assisted software development. It governs "repository truth" and execution alignment, ensuring that AI tools interact predictably and contribute to a stable, controlled project state.

The framework's core function is to manage the project's intent and ensure development aligns with it, rather than controlling the AI's internal processes. It enables a single repository instance to control exactly one project, providing clarity and preventing cross-project interference. The framework ensures precise handoffs and boundaries for execution tasks, minimizing scope creep and aligning work with defined intentions.
