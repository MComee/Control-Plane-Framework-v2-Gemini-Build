# Current Work Description

This file provides a human-readable description of the exact work item that is currently being addressed.

**Purpose:**
To offer a clear, natural language explanation of the task at hand, complementing the more formal `prompt.md` and `metadata.json` for human understanding.

**Content:**
This section should detail what the current work involves, its immediate objective, and any context necessary for a human to understand its purpose and scope.
This description should align with the active task defined in `project/priorities/now.md` and its corresponding file in `project/docs/tasks/`.
