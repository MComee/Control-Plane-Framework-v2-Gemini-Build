# Core Vision

This file defines the protected, fundamental doctrine of the controlled project. It outlines the ultimate purpose, long-term goals, and guiding principles that shape all development efforts.

**Guidance for Content:**
*   **Purpose:** Clearly articulate the "why" behind the project. What fundamental problem does it solve?
*   **Long-Term Goals:** Define aspirational outcomes and the overall direction of the project.
*   **Guiding Principles:** State the core values and philosophies that will underpin all decisions and development activities.
*   **Scope:** High-level boundaries of what the project aims to achieve.

This document is a cornerstone of repository truth and should only be updated through formal planning processes.
