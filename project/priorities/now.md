# Now

This file indicates the single, atomic task that is currently in progress or immediately designated for execution.

**Purpose:**
To provide a clear, unambiguous focus for the current development effort. This is the primary handoff point for execution tools.

**Content:**
This file should contain a direct reference (e.g., title or ID) to the active task definition located in `project/docs/tasks/`. The actual execution prompt and details will be found in `project/now/description.md`, `project/now/prompt.md`, and `project/now/metadata.json`.
