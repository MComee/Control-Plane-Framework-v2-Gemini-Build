# Blocked

This file lists tasks that cannot currently proceed due to external dependencies, unresolved issues, or other impediments.

**Purpose:**
To clearly identify and track work that is stalled, allowing for focused effort to resolve the blocking issues.

**Content:**
This file should contain a list of tasks (by title or ID) that are currently blocked. For each blocked task, a brief explanation of the blocking issue should be provided. Resolution of the blocking issue is required before the task can be moved out of this list.
