# Task Groups

This directory organizes implementation work into logical groups. Each task group bundles related atomic tasks that together contribute to the completion of a specific feature.

**Purpose:**
Task groups provide a higher-level organizational structure for development, making it easier to manage dependencies and track progress on larger pieces of work that span multiple atomic tasks.

**Guidelines for Task Group Files:**
*   Each file should follow the `framework/templates/task-group-template.md`.
*   Task group definitions are considered "Protected" artifacts and should not be modified by execution tools.
*   Task groups are tied to specific features and should be defined after features are established.
