# Definition of Done

Work is done only when all conditions below are true:

1. Task completion criteria in the task file are met.
2. Validation requirements in `project/now/metadata.json` are satisfied.
3. Required evidence is written to declared `project/evidence/` paths.
4. Forbidden and protected boundaries were not violated.
5. Planning and execution state remain synchronized to repository files.
6. Every documented path reference is real.
7. Root `README.md` recursive file tree exactly matches tracked repository structure.
